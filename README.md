# Институт вокальной психологии — сайт

Витрина Института вокальной психологии (ИВП). Реализована как маркетинговая витрина на Next.js;
доступ к материалам курсов ведётся через внешнюю LMS (выбирается командой ИВП).

## Стек

- Next.js 14 (App Router)
- React 18 + TypeScript
- Tailwind CSS + собственные UI-примитивы (в стиле shadcn/ui)
- React Hook Form + серверные экшены Next.js
- Resend для доставки писем с форм (по желанию — если задан `RESEND_API_KEY`)

## Локальный запуск

```bash
npm install
cp .env.example .env.local   # заполните при необходимости
npm run dev
```

Открыть http://localhost:3000

## Скрипты

- `npm run dev` — dev-сервер с hot-reload
- `npm run build` — прод-сборка
- `npm run start` — запуск прод-сборки
- `npm run typecheck` — проверка типов
- `npm run lint` — линтер

## Переменные окружения

См. `.env.example`:

- `RESEND_API_KEY` — ключ Resend для отправки писем с форм. Если не указан — заявки пишутся в консоль.
- `CONTACT_EMAIL` — куда доставлять заявки.
- `NEXT_PUBLIC_LMS_URL` — куда ведёт кнопка «Личный кабинет». Заглушка до выбора LMS.
- `NEXT_PUBLIC_YM_ID`, `NEXT_PUBLIC_GA_ID` — идентификаторы аналитики (подключение — в задачах после MVP).

## Что нужно от команды ИВП

Открытые пункты из раздела 12 брифа, которые нужно закрыть до релиза:

1. **Домен** — купить и указать в `siteConfig.url` (`lib/utils.ts`) и в `.env`.
2. **Юрлицо** — реквизиты для футера (`components/site/footer.tsx`), реальные тексты юр-страниц.
3. **Лицензия** — номер и скан для футера/страницы «Лицензия».
4. **Логотип** — заменить типографический (`components/site/logo.tsx`) на векторный при готовности.
5. **Фото Марии** — положить в `public/team/mariya.jpg` и заменить плейсхолдер в
   `app/team/mariya-osadchaya/page.tsx` и `app/about/page.tsx`.
6. **LMS** — указать реальный URL в `NEXT_PUBLIC_LMS_URL`.
7. **Финальные тексты** — вычитать и заменить рабочий копирайт на утверждённые формулировки.
